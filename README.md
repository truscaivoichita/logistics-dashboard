# Logistics Dashboard

A web application designed to simulate a simple logistics management system.
The project allows users to add delivery locations, visualize routes on an interactive map, and calculate distances between points.

This project combines my background in transport management with my journey into web development. It demonstrates how real-world logistics concepts can be translated into digital tools using JavaScript and mapping technologies.

The goal is to build a lightweight dashboard that helps visualize delivery routes and understand how route planning can improve efficiency in logistics operations.

## Features

- Add delivery points to the map
- Visualize routes between locations
- Calculate distance between delivery points
- Optimize route order for efficiency
- Display locations using an interactive map

## Technologies

- HTML5
- CSS3
- JavaScript
- Leaflet Map API (or Google Maps API)
- Geolocation API

## Project Goal

The purpose of this project is to practice JavaScript application development while building a tool related to logistics operations. It focuses on working with map APIs, handling geographic data, and building interactive user interfaces.

This project also reflects my professional experience in transport coordination, where route optimization and location management are key aspects of daily operations.
